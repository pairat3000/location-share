# 📍 Location Share

ระบบขอ Location จากผู้อื่น — ง่ายๆ ใน 1 คลิก

🌐 **Demo:** [https://pairat3000.github.io/location-share/](https://pairat3000.github.io/location-share/)

---

## วิธีการทำงาน

1. **ผู้ขอ** กรอกชื่อตัวเอง แล้วกด "สร้างลิงก์ส่งให้เพื่อน"
2. **ส่งลิงก์** ให้เพื่อนหรือคนที่ต้องการขอ location
3. **ผู้รับลิงก์** กด Allow location แล้วระบบจะเปิด Google Maps ให้ทันที

---

## Features

- สร้างลิงก์ขอ location แบบ unique สำหรับแต่ละคน
- เปิด Google Maps อัตโนมัติหลังได้รับ location
- รองรับ Mobile และ Desktop
- ไม่ต้อง login / ไม่เก็บข้อมูลใด

---

## Tech Stack

- HTML / CSS / JavaScript (Vanilla)
- Browser Geolocation API
- Hosted บน GitHub Pages
